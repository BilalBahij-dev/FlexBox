01 / Introduction

What Is Flexbox?

Flexbox is a one-dimensional layout model. You set a container to display: flex and its children automatically become flex items that you can align, distribute, and size with ease.

📦

Flex Container

The parent element. Apply display: flex here to activate flexbox for its direct children.

🧩

Flex Items

The direct children of the flex container. They follow the flex rules you define on the container (and optionally on themselves).

➡️

Main Axis

The primary direction items are laid out — horizontal by default (left → right). Controlled by flex-direction.

⬇️

Cross Axis

Perpendicular to the main axis — vertical by default. This is what align-items controls.

→ Main Axis
↓ Cross Axis

08 / Item Properties

flex-grow · flex-shrink · flex-basis

These three properties — set on flex items, not the container — control how items size themselves relative to available space.

Property What it does Default
flex-grow How much extra space an item takes up relative to siblings. 0 = don't grow. 1 = take equal share. 2 = take double the share of 1. 0
flex-shrink How much an item shrinks relative to siblings when space is tight. 0 = never shrink. 1 = shrink equally. 1
flex-basis The starting size of the item before grow/shrink is applied. Like width/height but smarter. auto

/* On flex items, not the container */
.item {
  flex-grow:   1;     /* grow to fill available space      */
  flex-shrink: 1;     /* allowed to shrink                 */
  flex-basis:  200px; /* start at 200px before growing     */
}

/* Shorthand: flex = grow shrink basis */
flex: 1;            /* flex: 1 1 0   — share space equally */
flex: 1 0 200px;   /* grow but don't shrink, start at 200px */
flex: 0 0 auto;    /* don't grow or shrink                  */
